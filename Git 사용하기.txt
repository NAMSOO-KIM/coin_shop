# Git 설정


 - Git 설치
   Git-2.31.1-64-bit.exe 실행

- Github에서 프로젝트 Fork 후 내 프로젝트에서 Clone 링크 복사
  https://github.com/for-ming/coin_shop.git  

- Git CMD 실행

- Git 사용할 폴더 생성 및 지정
  > cd C:\Users\COM\Desktop\git

- Git 프로젝트 받기
  > get clone https://github.com/for-ming/coin_shop.git

- 폴더 이동
  > cd C:\Users\COM\Desktop\git\coin_shop

- Origin 확인하기
  > git remote

- 합칠 팀원의 프로젝트 Upstream 지정
  > git remote add upstream https://github.com/NAMSOO-KIM/coin_shop
  > git remote

- Branch 명 보기
  > git branch

- 프로젝트 수정 후 변동 사항 확인하기
  > git status 	// 빨간색으로 변동 사항 표시

- 프로젝트 변동 사항 적용하기
  > git add .
  > git status	// 변동 사항이 초록색으로 표시

- 변동 사항 파일 설명하기
  > git commit -m "파일 설명할 내용"

- 내 프로젝트에 변동 사항 파일 올리기
  > git push origin main	// origin은 git remote에서 나온 내가 관리중인 프로젝트, main은 branch 의미

- 다른 팀원의 프로젝트에 변동 사항 파일 올리기
  > git push upstream main
